# TypeScript

## :one: Install TypeScript via `npm`

```
$ npm install typescript --global
```

* Test TypeScript

```
$ tsc --version
Version 3.7.4
```

## :two: REPL

```
$ npm install ts-node --global
```

* Test TS-NODE

```
$ ts-node --version
v8.5.4
```
