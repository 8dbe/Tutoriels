# Installation


## Téléchargement manuel (non recommandé)

:pushpin: Suivre les instructions pour installer manuellement 

https://nodejs.org/en/download/

## Par un `Package Manager`

:pushpin: Windows

```
> choco install nvm
```

:pushpin: MacOS


```
$ brew cask install nvm
```

## Installer node

```
$ nvm install 12
```

```
$ nvm use 12
```
