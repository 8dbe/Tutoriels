# AWS CLI

## :o: Installation

https://aws.amazon.com/fr/cli/

:m: Avec `package Managers`

:pushpin: [MacOS](https://formulae.brew.sh/formula/awscli)

```
$ brew install awscli
```

:pushpin: [Windows](https://chocolatey.org/packages/awscli)

```
$ choco install awscli
```

## :one: Configuration

```
$ aws config
```
